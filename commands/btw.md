# Better-OMO Workspace Tools

This command provides workspace management utilities for Better-OMO plugin.

## Available Actions

### `/btw status`
Show current workspace status including:
- Active sessions
- TMUX status
- Configuration summary

### `/btw config`
Display current Better-OMO configuration

### `/btw reset`
Reset workspace to default state:
- Clean up background tasks
- Reset session states
- Clear caches

## Examples

```bash
/btw status        # Check workspace status
/btw config        # View configuration
/btw reset         # Reset workspace
```

Use these tools to manage your development environment efficiently.
